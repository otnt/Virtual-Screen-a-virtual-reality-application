本程序需要CUDA支持，和Kinect配合使用。
如果你有Kinect，同时计算机支持CUDA库，那么可以进入../exe文件夹，通过shell调用格式：DIP_BASED_ON_CUDA.exe A B。其中A：0或者1，0表示B是一张图片，1表示B是一段视频。B：图片或视频的名字（需要包含后缀）。

本程序在Windows 7 64bit下经过运行测试。
